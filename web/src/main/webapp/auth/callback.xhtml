Información de integración

Esta es una página de retorno para el flujo OIDC. En una aplicación real, deberías:

  1. Extraer los parámetros code y state de la URL
  2. Enviarlos a GET /api/auth/gubuy/callback
  3. Recibir un token JWT e información del usuario
  4. Almacenar el token JWT en localStorage o una cookie
  5. Redirigir al área autenticada de tu aplicación

Ejemplo: El endpoint REST lo maneja automáticamente. Puedes probarlo accediendo a /api/auth/gubuy/callback?code=xxx&state=xxx en tu navegador o cliente de API.
