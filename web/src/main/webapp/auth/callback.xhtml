Integration Information

This is a callback page for the OIDC flow. In a real application, you would:

  1. Extract the code and state parameters from the URL
  2. Send them to GET /api/auth/gubuy/callback
  3. Receive a JWT token and user information
  4. Store the JWT token in localStorage or a cookie
  5. Redirect to the authenticated area of your application

Example: The REST endpoint handles this automatically. You can test it by accessing /api/auth/gubuy/callback?code=xxx&state=xxx in your browser or API client.
